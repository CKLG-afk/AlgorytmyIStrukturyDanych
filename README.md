## Zestaw programów zrobiony w ramach I semestru "Algorytmy i strutkury danych"

### Zrealizowane założenia:
* rozgałezienia warunkowe, pętle iteracyjne, rekurencja. Warunek stopu.
* Proste i złożone typy danych. Typy porządkowe. Tablica i rekord jako nie dynamiczne agregaty danych.
* Szacowanie złożoności obliczeniowej algorytmów.
* Listy liniowe. Algorytmy obsługi list liniowych: wyszukiwanie, wstawianie, usuwanie. Dynamiczne LIFO-stosy i FIFO-kolejki, listy dwukierunkowe, listy cykliczne. Drzewa binarne.

### Struktura repozytorium
W każdym katalogu jest kod źródłowy .cpp z zaimplementowanymi rozwiązaniami z zagadnień algorytmów
